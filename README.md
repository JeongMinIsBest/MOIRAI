## ‚úèÔ∏è Most Optimal Implementation to Reinforce Academic Instruction
**MOIRAI** is an **AI-powered example problem generation service** designed to **enhance users‚Äô learning effectiveness**.
  
By applying **In-context Learning** to a **well-pretrained Large Language Model (LLM)**, MOIRAI generates example problems tailored to the learning topics specified by the user.
  
This project integrates:
- **Experiential Learning Theory** by *David A. Kolb*
- The **Havruta (Hevruta) learning method** from Jewish educational traditions
  
Through AI-generated problems, users can repeatedly apply learned concepts, validate their understanding in practical contexts, and reinforce theoretical knowledge.

### References
- Experiential Learning ‚Äî David A. Kolb  
- https://j-kagedu.or.kr/upload/pdf/kagedu-10-3-73.pdf

## üìå Features

### Main Screen
- Users input the topic and main subject they want to study  
- By clicking the **Generate Problem** button, problems related to the input topic are generated  
- Due to the long text-based nature of learning content, a **Clear Text** button is provided  
- The **Bookmark** button (top-right) allows users to view saved problems

### Problem Generation & Review Screen
- Users navigate to this screen by clicking **Generate Problem** on the main screen  
- Problems are generated based on user input  
- Users can check answers using the **Show Answer** button  
- Clicking **Generate New Problem** creates a different problem  
- **Save Current Problem** allows bookmarking of the displayed problem

- Displays problems bookmarked by the user  
- Implemented as a scroll view for reviewing multiple problems and answers
